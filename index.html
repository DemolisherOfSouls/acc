ACC++

ACC++ is an ACS script compiler for use with ZDoom and Hexen. It extends ACC and implements C++ features, replacing C code.

Download this project as a .zip file Download this project as a tar.gz file

What is ACC++?

ACC++ adds many new features and error checking additions, including comprehensive warnings, custom data types, structs, and true preprocessor directives. It allows scripts to be written easier, and still generates ZDoom compatible byte code.

Preprocessor Directives

You remember #define, and #include, but they were always executed in order alongside the actual code, making advanced tasks difficult. Now they are parsed before the code, allowing macros, and the ability to define properties within a library, and have them apply to the first line of code, no matter where they are placed, even if they are placed within a script:

#define DEF_VAL 3
script 231 enter
{
#define MACRO int x = DEF_VAL
    MACRO;
    SetResultValue(x);
}

Stricter Types

A bool is now just that. A bool. It has a value of 1 or 0. The compiler will throw warnings or errors depending on your command line parameters.

string s = "d";

int i = s; //Warning: typedef 'string' assigned to typedef 'int'

New Keywords

A few new keywords have been defined, however this was a necessary evil. The new features should not impact most scripts, as the more advanced functions must be explicitly included, to avoid redefinition. Some new ones include enum, struct, and typedef.
